# Проверка module_2

Скрипт запускается на HQ-CLI и проверяет задания с 1 по 13. Пункт 14 выводится как `N/A`.

Проверьте адреса и учётные данные в `.env`, затем:

```bash
chmod +x ./01-check-module-2.sh
su -
cd /путь/module_2_test
./01-check-module-2.sh
```

При отсутствии диагностических утилит скрипт установит `openssh-clients`, `sshpass`, `expect`, `curl` и `bind-utils`.

Проверяемые службы не изменяются. Временный Expect-файл удаляется после завершения.

Проверить только формат итоговой таблицы без сетевых подключений:

```bash
./01-check-module-2.sh --self-test
```
